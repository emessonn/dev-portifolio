# Developer Portfolio

A modern, responsive developer portfolio web application built with React and Vite.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Clean, gradient-based design with smooth animations
- **Smooth Scrolling** - Navigate sections with smooth scroll behavior
- **Interactive Components** - Engaging hover effects and transitions
- **Dark Theme** - Eye-friendly dark color scheme with purple/blue gradients
- **Performance Optimized** - Built with Vite for lightning-fast development and builds

## 📋 Sections

- **Hero** - Eye-catching introduction with animated code block
- **About** - Personal information and statistics
- **Projects** - Showcase of your work with technology tags
- **Skills** - Technology proficiency with visual progress bars
- **Contact** - Contact form and social links
- **Footer** - Copyright and navigation

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open your browser and visit: `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The optimized production build will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## 🎨 Customization

### Personal Information

Update the following files to add your personal information:

- **Hero Section**: [src/components/Hero.jsx](src/components/Hero.jsx)
- **About Section**: [src/components/About.jsx](src/components/About.jsx)
- **Projects**: [src/components/Projects.jsx](src/components/Projects.jsx)
- **Skills**: [src/components/Skills.jsx](src/components/Skills.jsx)
- **Contact**: [src/components/Contact.jsx](src/components/Contact.jsx)

### Styling

- Global styles: [src/index.css](src/index.css)
- Component-specific styles are in their respective `.css` files

### Colors

The main color scheme uses:

- Primary: `#667eea` (Purple-Blue)
- Secondary: `#764ba2` (Purple)
- Background: `#1a1a2e` to `#0f3460` (Dark Blue Gradient)

To change colors, update the CSS variables and gradient values in the component CSS files.
